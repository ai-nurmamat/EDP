# TOOLS.md - 工具说明

## GitHub CLI 常用命令

```bash
# 查看开放 Issue
gh issue list --state open --repo <repo>

# 分配 Issue
gh issue edit <number> --assignee <user>

# 添加标签
gh issue edit <number> --label "bug,high-priority"
```
